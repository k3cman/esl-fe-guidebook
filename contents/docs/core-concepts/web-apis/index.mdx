
## Core Web APIs in JavaScript

### What are Web APIs?
Web APIs are interfaces provided by the browser or runtime environment (e.g., Node.js) to interact with the underlying system, the browser, or the internet. These APIs enable developers to manipulate documents, handle events, interact with servers, and much more.

### 1. Document Object Model (DOM)
The DOM API allows developers to interact with and manipulate the HTML and CSS of a webpage.

Common Methods:
Selecting elements:

```javascript
const element = document.getElementById('myElement');
const elements = document.querySelectorAll('.myClass');
```
Manipulating elements:
```javascript
const heading = document.getElementById('myHeading');
heading.textContent = 'Hello World!';
heading.style.color = 'blue';
```
Creating new elements:
```javascript
const newElement = document.createElement('div');
newElement.textContent = 'I am a new element!';
document.body.appendChild(newElement);
```

### 2. Fetch API
The Fetch API provides an easy way to make HTTP requests.

Example:

```javascript
fetch('https://api.example.com/data')
  .then(response => {
    if (!response.ok) {
      throw new Error('Network response was not ok');
    }
    return response.json();
  })
  .then(data => console.log(data))
  .catch(error => console.error('Fetch error:', error));
Using async/await:
async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Fetch error:', error);
  }
}

fetchData();
```

### 3. Geolocation API
The Geolocation API retrieves the geographic position of the user's device.

Example:
```javascript
navigator.geolocation.getCurrentPosition(
  position => {
    console.log('Latitude:', position.coords.latitude);
    console.log('Longitude:', position.coords.longitude);
  },
  error => console.error('Error getting location:', error)
);
```

### 4. History API
The History API allows manipulation of the browser's session history (e.g., navigating between pages programmatically).

Example:
```javascript
Navigate programmatically:
history.pushState({}, 'New Page', '/new-page');
console.log('Current URL:', window.location.href);
Listen for history changes:
window.onpopstate = () => {
  console.log('URL changed:', window.location.href);
};
```

### 5. Canvas API
The Canvas API is used to draw graphics on a webpage.

Example:
```javascript
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

// Draw a rectangle
ctx.fillStyle = 'blue';
ctx.fillRect(50, 50, 100, 100);

// Draw a circle
ctx.beginPath();
ctx.arc(150, 150, 50, 0, Math.PI * 2);
ctx.fillStyle = 'red';
ctx.fill();
```


### 6. Web Storage API
The Web Storage API allows storing data in the browser.

Local Storage:
Set an item:
```javascript
localStorage.setItem('username', 'Alice');
```
Get an item:
```javascript
const username = localStorage.getItem('username');
console.log(username); // 'Alice'
```
Remove an item:
```javascript
localStorage.removeItem('username');
```
Session Storage:
Works like local storage but only for the session.
```javascript
sessionStorage.setItem('token', 'abc123');
```

### 7. Notifications API
The Notifications API allows web applications to send system-level notifications.

Example:
```javascript
if ('Notification' in window) {
  Notification.requestPermission().then(permission => {
    if (permission === 'granted') {
      new Notification('Hello!', {
        body: 'You have a new message.',
        icon: '/icon.png'
      });
    }
  });
}
```

### 8. WebSockets API
WebSockets enable two-way communication between the client and server.

Example:
```javascript
const socket = new WebSocket('wss://example.com/socket');

// Listen for messages
socket.addEventListener('message', event => {
  console.log('Message from server:', event.data);
});

// Send a message
socket.send('Hello, server!');
```

### 9. File API
The File API allows reading and uploading files.

Example:
```javascript
const input = document.getElementById('fileInput');
input.addEventListener('change', event => {
  const file = event.target.files[0];
  const reader = new FileReader();

  reader.onload = () => {
    console.log('File content:', reader.result);
  };

  reader.readAsText(file);
});
```

### 10. Intersection Observer API
The Intersection Observer API detects when elements come into view.

Example:
```javascript
const observer = new IntersectionObserver(entries => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      console.log('Element is in view:', entry.target);
    }
  });
});

const element = document.getElementById('myElement');
observer.observe(element);
```
