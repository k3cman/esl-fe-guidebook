
# Core JavaScript Concepts

## Variables and Data Types

JavaScript you should know:

- **Primitive Types**: `string`, `number`, `boolean`, `null`, `undefined`, `symbol`, `bigint`
- **Reference Types**: `object`, `array`, `function`, `Map`,`Set`

Variables can be declared using `var`, `let`, or `const`:

```javascript
let name = 'Alice';
const age = 30;
var isActive = true;
```

Functions
Functions are reusable blocks of code that perform a specific task. They can be declared using function declarations or function expressions:

```javascript

// Function Declaration
function greet(name) {
  return `Hello, ${name}!`;
}

// Function Expression
const greet = function(name) {
  return `Hello, ${name}!`;
};

// Arrow Function
const greet = (name) => `Hello, ${name}!`;
For more information, see MDN Web Docs: Functions.
```
Scope and Closures
JavaScript has function scope and block scope. Closures allow a function to access variables from its outer scope even after the outer function has returned:

```javascript
function outerFunction() {
  let outerVariable = 'I am outside!';

  function innerFunction() {
    console.log(outerVariable);
  }

  return innerFunction;
}

const myFunction = outerFunction();
myFunction(); // Logs: I am outside!
```
For more information, see MDN Web Docs: Closures.

Objects and Arrays
Objects are collections of key-value pairs, and arrays are ordered lists of values:

```javascript
// Object
const person = {
  name: 'Alice',
  age: 30,
  greet() {
    console.log('Hello!');
  }
};

// Array
const numbers = [1, 2, 3, 4, 5];
```
For more information, see MDN Web Docs: Working with objects and MDN Web Docs: Arrays.

Promises and Asynchronous Programming
Promises represent the eventual completion (or failure) of an asynchronous operation:

```javascript
const fetchData = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve('Data fetched');
  }, 2000);
});

fetchData
  .then(data => console.log(data))
  .catch(error => console.error(error));
async/await provides a more readable way to work with promises:
```
```javascript
async function fetchData() {
  try {
    const data = await fetch('https://api.example.com/data');
    const result = await data.json();
    console.log(result);
  } catch (error) {
    console.error(error);
  }
}

fetchData();
```

Event Handling
JavaScript can handle events such as clicks, form submissions, and more:

```javascript
Edit
document.getElementById('myButton').addEventListener('click', () => {
  console.log('Button clicked!');
});
```

Modules
Modules allow you to organize code into reusable pieces:

```javascript
// Exporting a module
export const greet = (name) => `Hello, ${name}!`;

// Importing a module
import { greet } from './greet.js';
console.log(greet('Alice'));
```
